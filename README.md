### 概要
Jetpack Composeの学習の一環で作成した本の一覧表示画面UIサンプルです。
ECサイト風のUIを意識して作成しました。

各本のデータは[supabase](https://supabase.com/)と呼ばれるBaaS (バックエンドサービス)からリモートで取得し、画面に表示しています。
